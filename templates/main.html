This is a tool design to help new SW employees quickly obtain datasheet information on products.

The tool uses LLM technology to gather info from the sheets and display them in a conversational format.

The tool is still is very basic in its current state and has access to very little products.

The tool is planned to be updated as a functioning chat, a la chatGPT, but for paint. In its current state it only acts as a Q and A bot.
